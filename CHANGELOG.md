# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-01 (Beta)

### Added

- Initial public beta release
- Real-time deployment tracking via Coolify API and database
- BullMQ queue monitoring with job inspection
- PostgreSQL and PgBouncer metrics via Prometheus
- VPS system metrics (CPU, memory, disk, load)
- Site health monitoring with SSL checks
- Uptime Kuma integration
- AutoHEAL system for automatic app recovery
- Simple password authentication
- Dark mode UI with Tailwind CSS
